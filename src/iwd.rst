 iwd

SYNOPSIS
========

**iwd** [*options* ...]

DESCRIPTION
===========

Daemon for managing Wireless devices on Linux.

OPTIONS
=======

--version, -v           Show version number and exit.
--help, -h              Show help message and exit.
